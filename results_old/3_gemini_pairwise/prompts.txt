PAIRWISE – ZERO-SHOT PROMPT (create_pairwise_zero_shot_prompt)
--------------------------------------------------------------------------------

Compare these two user stories and determine which requires more development effort:

**STORY A:**
{story_a}

**STORY B:**
{story_b}

**ANALYSIS FRAMEWORK:**
- Technical complexity (algorithms, integrations, data processing)
- Implementation scope (UI changes, backend logic, database modifications)
- Testing requirements (unit tests, integration tests, user acceptance)
- Risk factors (dependencies, unknowns, potential issues)

**IMPORTANT: You must choose one story as requiring more effort. Equal complexity is very rare in software development.**

**OUTPUT FORMAT:**
Analysis: [brief comparison of complexity factors]
Decision: [1 if Story A requires more effort, -1 if Story B requires more effort]
Confidence: [0.0 to 1.0]
